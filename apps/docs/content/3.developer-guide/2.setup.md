# Setting Up Development Environment

## Prerequisites

- Node.js (v24+)
- pnpm (v8+)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gustawdaniel/vault-track.git
   cd vault-track
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

## Project Structure

- `apps/ui`: The main Vue/Nuxt application.
- `apps/docs` (legacy), `apps/my-docs` (new): Documentation sites.
- `libs/qif2json`: Library for parsing QIF files.
- `libs/qif-ts`: TypeScript types for QIF data.

## Environment Variables

Copy `.env.example` to `.env` in `apps/ui` if necessary.

- `REDIS_URL`: URL for the Redis instance (optional, defaults to FS storage).
